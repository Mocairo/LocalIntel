# 阅读工作流再设计执行计划

## 成功标准

- 首页右侧摘要不再形成单列长瀑布。
- 近期趋势图更醒目，数据点不足时仍能展示 7 天趋势。
- 今日情报重点排序和分类列表默认三列展示。
- 每张情报卡片高度固定，长内容不撑开整行。
- 详情抽屉能承接完整阅读和标记动作。

## 步骤

1. 更新布局测试
   - 增加首页平衡网格、趋势主图、今日队列摘要、详情抽屉和三列卡片断言。
   - 验证：先运行目标测试并确认失败。

2. 调整首页摘要区
   - 新增 `overview-balance-grid`。
   - 让趋势卡横跨两列，并改为更醒目的面积折线图。
   - 增加 `paddedTrendRows`，在历史数据不足时补齐展示点。

3. 调整今日队列
   - 新增 `renderQueueSummary` 和 `todayQueueSummary`。
   - 将情报卡片默认排布改为三列。
   - 固定卡片高度并限制标题、摘要、判断、标签和元信息溢出。

4. 强化详情抽屉
   - 重构详情内容为标题动作区、正文区、判断区和相关条目区。
   - 支持在抽屉内直接稍后看、归档、收藏、忽略。

5. 验证
   - 运行 `python -m pytest tests\test_dashboard_layout.py -q`。
   - 运行 `python -m pytest -q`。
   - 运行浏览器预览检查关键页面。
   - 运行 `git diff --check`。
